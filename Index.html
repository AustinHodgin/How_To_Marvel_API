Introduction

What is an API? Why would I use an API? How do I find out what type of API’s are out there? You might find your self asking these questions when you start getting interested in web development. API stands for Application Program Interface. You can think of this is a free resource that works in the background of websites. This resource will allow you to connect to other computers to pull or gather information and then display or use that information in your websites.

How Do We Use Them?

You might be thinking, “Wow, API’s sound cool. Where can I find them and how do I use them?” Well, if you are interested in where to find some really cool API’s then I would suggest going here. This is a list of a bunch of diffrent API's that you can use in your website to make some cool things.

In this How To Guide I will show you how to use the Marvel API to add comic information to your website. This can range from pulling the covers of comics to pulling information about characters and story arcs. If you would like to see the documentation from Marvel about their API you can find it here.
